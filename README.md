# R Statistics Visualization

This repository contains R scripts for statistical data visualization techniques, covering fundamental plotting methods for univariate data analysis. The code demonstrates both base R graphics and ggplot2 implementations for creating dotplots (stripcharts), histograms, stem-and-leaf plots, and boxplots. Each script includes practical examples with customizable parameters for creating professional statistical visualizations, making it a valuable resource for learning R programming and data visualization fundamentals.
